## installation

```yaml
prime_icons: ...
```
## Usage

```dart
import 'package:prime_icons/prime_icons.dart';

class MyWidget extends StatelessWidget {
  Widget build(BuildContext context) {
    return Center(
      child: Icon(PrimeIcons.lock),
     );
  }
}
```
